Return the MCDO standard deviation (i.e. the square root of the diagonal covariance) for sample sample_idx from run_path.

Parameters:
run_pathstr

Path of the hydra run.

sample_idxint

Sample index.

patch_idx_listlist of int or str, optional

Patch indices to restrict to. Must be a subset of cfg.inference.patch_idx_list of the original run. If a string, bayes_dip.utils.experiment_utils.get_predefined_patch_idx_list(patch_idx_list) is used. If None (the default), all patch indices are used.

subtract_image_noise_correction_if_anybool, optional

Whether to subtract the image noise correction term (if any) from the covariance diagonal before taking the square root. The default is True.

experiment_pathsdict, optional

See translate_path().

Returns:
stddevTensor

Standard deviation. Shape: (im_size, im_size).
