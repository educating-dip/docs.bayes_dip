bayes_dip.data.trafo.parallel_beam_2d_ray_trafo module

Provides ParallelBeam2DRayTrafo, as well as getters for its matrix representation and a MatmulRayTrafo implementation.

class bayes_dip.data.trafo.parallel_beam_2d_ray_trafo.ParallelBeam2DRayTrafo(im_shape: Tuple[int, int], num_angles: int, first_angle_zero: bool = True, angular_sub_sampling: int = 1, impl: str = 'astra_cuda')[source]

Bases: BaseRayTrafo

Ray transform implemented via ODL.

Adjoint computations use the back-projection (might be slightly inaccurate).

Attributes:
angles

np.ndarray : The angles (in radian).

Parameters:
im_shape2-tuple of int

Image shape, (im_0, im_1).

num_anglesint

Number of angles (to distribute from 0 to pi).

first_angle_zerobool, optional

Whether to shift all angles such that the first angle becomes 0.. If False, the default angles from ODL are used, where the first angle is at half an angle step. The default is True.

angular_sub_samplingint, optional

Sub-sampling factor for the angles. The default is 1 (no sub-sampling).

implstr, optional

Backend for odl.tomo.RayTransform. The default is 'astra_cuda'.
