Bases: BaseNeuralBasisExpansion

Partial implementation of BaseNeuralBasisExpansion using matmul with the assembled Jacobian matrix for the jvp() and vjp() methods.

The Jacobian matrix property matrix and methods assembling it (i.e., get_matrix() and update_matrix()) are abstract in this class.

Class for Jacobian vector products and vector Jacobian products.

This class stores all the stateful information needed for these operations as attributes and exposes just the jvp() and vjp() methods.
