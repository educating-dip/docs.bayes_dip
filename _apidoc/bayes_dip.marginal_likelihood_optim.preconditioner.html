bayes_dip.marginal_likelihood_optim.preconditioner module

Provides preconditioners for applying CG to the linear system of the observation covariance.

class bayes_dip.marginal_likelihood_optim.preconditioner.BasePreconditioner(**update_kwargs)[source]

Bases: ABC

Base preconditioner class.

The preconditioner commonly is an approximation of the system matrix, and its inverse is applied to the residuals to solve an equivalent system with improved condition with conjugate gradients.

The closure returned by get_closure() multiplies with the inverse (like needed for preconditioning), i.e. it performs a matmul() with use_inverse=True.

Return a preconditioner for observation covariance.

Parameters:
kwargsdict

Keyword arguments. kwargs['name'] is required, options are:

  • kwargs['name'] == 'low_rank_eig'

    Return a LowRankObservationCovPreconditioner instance. Further (required) keyword arguments are 'batch_size', 'low_rank_rank_dim' and 'oversampling_param' (see LowRankObservationCovPreconditioner).

  • kwargs['name'] == 'incomplete_chol'

    Return an IncompleteCholeskyPreconditioner instance. Further (required) keyword arguments are 'batch_size', 'low_rank_rank_dim' and 'approx_diag_num_samples' (see IncompleteCholeskyPreconditioner).
