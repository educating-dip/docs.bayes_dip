bayes_dip.utils.evaluation_utils module

Utilities for evaluation.

A general note on hydra run paths: the methods in this module expect paths to single hydra output folders (containing the .hydra/ subfolder), which may either be a normal hydra output path or a path to a sub-folder of a hydra multirun path (such sub-folders are called “multirun path” in this module’s documentation).
