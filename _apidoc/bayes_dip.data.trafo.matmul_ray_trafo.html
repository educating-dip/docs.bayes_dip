Provides MatmulRayTrafo.

class bayes_dip.data.trafo.matmul_ray_trafo.MatmulRayTrafo(im_shape: Union[Tuple[int, int], Tuple[int, int, int]], obs_shape: Union[Tuple[int, int], Tuple[int, int, int]], matrix: Union[Tensor, spmatrix, ndarray], fbp_fun: Optional[Callable[[Tensor], Tensor]] = None, angles: Optional[ArrayLike] = None)[source]

Bases: BaseRayTrafo

Ray transform implemented by (sparse) matrix multiplication.

Adjoint computations are accurate in this implementation (which is not always the case when using back-projection for the adjoint).

Attributes:
angles

array-like : The angles (in radian).
