Covariance in observation space using low-rank matrix approximation. The algorithm is described in [1].

get_low_rank_observation_cov_basis(use_cpu: bool = False, eps: float = 0.1, verbose: bool = True, batch_size: int = 1)[source]ÔÉÅ

Eigenvalue Decomposition in One Pass.

This method implements Algo. 5.6 from Halko et al. and computes an approximate eigenvalue decomposition of the low-rank term of covariance in observation space.

Parameters:

Whether to compute QR on CPU. The default is False.

Minimum value eigenvalues. The default is 1e-3.

verbosebool, optional

If True, print eigenvalue information. The default is True.
