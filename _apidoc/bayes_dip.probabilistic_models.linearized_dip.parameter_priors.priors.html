class bayes_dip.probabilistic_models.linearized_dip.parameter_priors.priors.BaseGaussPrior(init_hyperparams: Dict[str, Any], modules: Sequence[Conv2d], device: Union[str, device])[source]

Bases: Module, ABC

Base Gaussian prior class for nn.Conv2d modules.

The prior is placed over all filters of modules (passed to __init__()), i.e. all input-channel-output-channel combinations of all modules share the prior.

Implements efficient batched multiplication with the covariance matrices of multiple priors of the same type via the class method batched_cov_mul().

classmethod batched_cov_mul(priors: Sequence, v: Tensor, use_cholesky: bool = False, use_inverse: bool = False, **cov_mat_kwargs) → Tensor[source]

Multiply weight vectors with a stack of covariance matrices.

Parameters:
priorssequence of BaseGaussPrior

Sequence of priors of the same type. The covariance matrix of each prior is repeated prior.num_total_filters times, and the matrices of all priors are then concatenated.

vTensor

Batch of flattened and concatenated weight parameters of Conv2d modules. Shape: (batch_size, n * kernel_size ** 2), where n is the sum of the numbers of filters of all modules of all priors.

use_choleskybool, optional

If True, multiply with the Cholesky factor instead. Cannot be combined with use_inverse. The default is False.

use_inversebool, optional

If True, multiply with the inverse instead. Cannot be combined with use_cholesky. The default is False.

cov_mat_kwargsdict, optional

Keyword arguments passed to cov_mat() of each prior (e.g. an eps value).

Returns:
v_cov_mulTensor

Products. Has the same shape as v.
